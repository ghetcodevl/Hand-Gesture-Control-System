# Hand Gesture Control System

## 📌 Overview
This project is a real-time hand gesture control system using computer vision.

## 🚀 Features
- Hand gesture recognition using MediaPipe
- Cursor control using gestures
- Zoom in/out using hand movements
- Switch drawing mode using gestures

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
